monome 2011 256 grid visualizer

a real-time audio visualizer for the monome grid. audio input is analyzed and mapped to the grid's leds, giving you a live visual response to whatever is playing through your system.

originally built around the 2011 monome grid (16×16, 4-step varibright), it works with any monome grid supported by serialosc. eight visual presets are included: spectrum analyzers, a lissajous XY plot, a beat pulse detector, a flame-style waveform, a symmetric center waveform, and carassius auratus — three fish swimming freely across the grid, each driven by a different frequency band. each preset has its own gain setting and the spectrum presets offer per-band and per-column brightness controls.

controlled entirely through a browser UI at http://localhost:5000. runs quietly in the background with a system tray icon. works on windows, macOS, and linux.
